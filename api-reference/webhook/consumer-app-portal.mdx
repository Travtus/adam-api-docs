---
title: "Consumer Application Portal"
api: "GET  https://api.travtus.com/webhook/app-portal-access/"
description: "This endpoint allows users to request a short-lived URL for accessing consumer application portal."
version: "v2"
---


### Header

<ParamField header="Authorization" type="string" default="none" required>
    The authentication token for your request
</ParamField>

### Response
<ResponseField name="url" type="string">
    The short-lived URL for accessing your webhook consumer application portal.
</ResponseField>

<RequestExample>

    ```bash Example Request
    curl --location --request GET 'https://api.travtus.com/webhook/app-portal-access/' \
    --header 'Authorization: Bearer <token>' \
    --data ''
    ```

</RequestExample>
