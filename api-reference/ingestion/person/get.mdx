---
title: "Get Person"
api: "GET https://api.travtus.com/persons?externalref={externalref}&identifier={identifier}/"
description: "This endpoint retrieves infromation about one person with the provided identifier."
---

You need to provide either the external reference (`externalref`) OR the internal id (`identifier`) in order to successfully retrieve the person record.

### Header

<ParamField header="Auhtorization" type="string" default="none" required>
  The authentication token for your request.
</ParamField>

### Query Parameters

<ParamField query="externalref" type="string" default="none" required>
  The external reference for the person to be retrieved.
</ParamField>

<ParamField query="identifier" type="string" default="none" required>
  The internal identifier for the person to be retrieved
</ParamField>

### Response

<ResponseField name="email_address" type="string">
  The e-mail address of the retrieved person.
</ResponseField>

<ResponseField name="first_name" type="string">
  The first name of the retrieved person.
</ResponseField>

<ResponseField name="last_name" type="string">
  The last name of the retrieved person.
</ResponseField>

<ResponseField name="external_ref" type="string">
  The external reference of the retrieved person. This is unique per portfolio.
</ResponseField>

<ResponseField name="identifier" type="string">
  The internal id of the retrieved person. This is entirely unique across all companies and portfolios.
</ResponseField>

<ResponseField name="phone_number" type="string">
  The phone number of the retrieved person.
</ResponseField>

### Errors
Listed below are common errors that may be returned by the endpoint, along with their corresponding status code.

Status Code - 400
```json Missing Authorization header
{
   "error": {
    "type": "missing_authorization",
    "message": "Your request does not include an 'Authorization' header with a bearer token for your account."
  }
}
```

Status Code - 401
```json Invalid/Expired Authorization header bearer token value
{
   "error": {
    "type": "invalid_authorization",
    "message": "The bearer token you have provided in the 'Authorization' header is invalid."
  }
}

```

Status Code - 400

Returned if neither the `externalref` nor the `identifier` query params are passed in the request.
```json No identifier or external reference query param provided
{
   "error": {
    "type": "no_identifier",
    "message": "Please provide either an externalref or an internal identifier."
  }
}

```

Status Code - 404

Returned if a person record with the provided `externalref` or `identifier` could not be found.
```json No person record found for the provided externalref or identifier
{
   "error": {
    "type": "not_found",
    "message": "There is no person record with the provided external reference or internal id."
  }
}

```
