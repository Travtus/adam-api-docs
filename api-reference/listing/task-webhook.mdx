---
title: "Create Task Webhook Event"
api: "POST https://www.example.com/webhook/task_creation"
description: "This endpoint for receiving task creation webhook event."
---

This endpoint allows you to receive a task creation event from Adam. Please
refer to [Webhook Subscription Page](/api-reference/webhook/subscription) to subscribe
the webhook event you want to receive.

When you receive a task creation event, you must send a 200 response.
If you do not send a 2xx response Adam will resend a task creation event until 3 times failure.

### Header

<ParamField header="Authorization" type="string" default="none" required>
  The authentication token for your request
</ParamField>

### Errors

Listed below are common errors that may be returned by the endpoint, along with their corresponding status code.

Status Code - 400
```json Missing Authorization header
{
   "error": {
    "type": "missing_authorization",
    "message": "Your request does not include an 'Authorization' header with a bearer token for your account."
  }
}
```

Status Code - 401
```json Expired Authorization header bearer token value
{
   "error": {
    "type": "expired_token",
    "message": "The bearer token you have provided in the 'Authorization' header has expired. Please obtain a new one."
  }
}
```

Status Code - 401
```json Invalid Authorization header bearer token value
{
   "error": {
    "type": "invalid_authorization",
    "message": "The bearer token you have provided in the 'Authorization' header is invalid."
  }
}
```
