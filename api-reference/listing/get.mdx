---
title: "Get Listings By Group"
api: "GET https://api.travtus.com/listings/{groupid}?after-id={after-id}/"
description: "This endpoint retrieves listings and unit IDs for a specific group."
---

### Header

<ParamField header="Authorization" type="string" default="none" required>
  The authentication token for your request.
</ParamField>

### Query

<ParamField query="after-id" type="string" default="1">
  The API returns a maximum of 10 records per call.
  
  You can use this query parameter to load the next set of records by passing in the value returned in the response, under `last_id`.
</ParamField>

### Path Parameters

<ParamField path="groupid" type="string" default="none" required>
  The id of the group you want to retrieve listings for.
</ParamField>

### Response
<ResponseField name="listings" type="[object]">
  A list of retrieved listings for the provided group id and their corresponding units.
  <Expandable title="listings">
    <ResponseField name="listing_id" type="string">
      The internal ID of the retrieved listing.
    </ResponseField>
    <ResponseField name="unit_id" type="string">
      The intenral ID of the retrieved unit for the listing
    </ResponseField>
  </Expandable>
</ResponseField>

### Errors

Listed below are common errors that may be returned by the endpoint, along with their corresponding status code.

Status Code - 400
```json Missing Authorization header
{
   "error": {
    "type": "missing_authorization",
    "message": "Your request does not include an 'Authorization' header with a bearer token for your account."
  }
}
```

Status Code - 401
```json Invalid/Expired Authorization header bearer token value
{
   "error": {
    "type": "invalid_authorization",
    "message": "The bearer token you have provided in the 'Authorization' header is invalid."
  }
}
```

Status Code - 401

Returned if the currently authenticated API user does not have access to the provided group
```json No access to the provided group
{
   "error": {
    "type": "no_access_to_group",
    "message": "You do not have access to the group you have provided."
  }
}
```

Status Code - 404

Returned if no listings are found for the provided group id.
```json No listing records found for the provided group id
{
   "error": {
    "type": "not_found",
    "message": "There are no listing records for the provided group id."
  }
}
```
