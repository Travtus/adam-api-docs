## üéØ **Key Dashboard Widgets**

### **Portfolio Overview**
<Accordion title="Occupancy Metrics">
- **Current Occupancy Rate**: Percentage of occupied units across all properties
- **Available Units**: Number of units ready for immediate occupancy
- **Units in Transition**: Units being prepared for new residents
- **Maintenance Hold**: Units temporarily unavailable due to maintenance
- **Occupancy Trend**: 30, 60, and 90-day occupancy trends
</Accordion>

<Accordion title="Financial Summary">
- **Monthly Revenue**: Current month's rental income and projections
- **Outstanding Balances**: Total amount owed by residents
- **Collection Rate**: Percentage of rent collected on time
- **Expense Tracking**: Major expense categories and budget vs. actual
- **Net Operating Income**: Revenue minus operating expenses
</Accordion>

<Accordion title="Maintenance Overview">
- **Open Work Orders**: Total number of pending maintenance requests
- **Emergency Requests**: High-priority items requiring immediate attention
- **Average Resolution Time**: How quickly maintenance issues are resolved
- **Vendor Performance**: Completion rates and response times by vendor
- **Preventive Maintenance**: Scheduled maintenance items coming due
</Accordion>

<Accordion title="Resident Activity">
- **New Applications**: Recent lease applications and their status
- **Move-ins/Move-outs**: Scheduled resident transitions
- **Communication Activity**: Recent messages, announcements, and responses
- **Satisfaction Scores**: Resident feedback and satisfaction ratings
- **Lease Renewals**: Upcoming lease expirations and renewal status
</Accordion>

## üÜò **Troubleshooting Dashboard Issues**

### **Common Issues**

<Accordion title="Data Not Loading">
**Symptoms**: Widgets show loading indicators or no data
**Solutions**:
- Check your internet connection
- Refresh the page (Ctrl+F5 or Cmd+Shift+R)
- Clear your browser cache
- Contact support if issues persist
</Accordion>

<Accordion title="Incorrect Data">
**Symptoms**: Numbers don't match expectations or other reports
**Solutions**:
- Verify date range and property filters
- Check data sync status in settings
- Review recent data imports or changes
- Compare with source systems
</Accordion>

<Accordion title="Slow Performance">
**Symptoms**: Dashboard loads slowly or is unresponsive
**Solutions**:
- Reduce number of widgets on dashboard
- Use shorter time periods for historical data
- Close other browser tabs
- Try using a different browser
</Accordion>
