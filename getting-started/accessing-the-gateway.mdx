---
title: "Accessing the Gateway"
description: "Learn how to access the Gateway"
---

The Gateway can be accessed in one of a few ways:\
​\
​**Google Chrome Extension**

1. Visit the Chrome Web Store or [<u>Travtus for Google Chrome</u>](https://chromewebstore.google.com/detail/travtus-gateway/clncechdmbedjkkfmpmjlbbfkgkhmemk)
2. Search for "Travtus" in the Chrome Web Store.
3. When you find an extension you'd like to add, click Add to Chrome.

\
**Outlook Extension**

1. Visit [<u>Travtus in Microsoft AppSource</u>](https://appsource.microsoft.com/en-us/product/office/WA200006294?tab=Overview)
2. Select 'Get it now'.
3. Follow the instructions to download and install the app.

**Microsoft Edge**

1. In Microsoft Edge, go to the [<u>Chrome Web Store</u>](https://chromewebstore.google.com/category/extensions).
2. Select **Allow extensions from other stores** in the banner at the top of the page, then select Allow to confirm.
3. Select the extension you want to add and select **Add to Chrome**.
4. At the prompt showing permissions required by the extension, carefully review the permissions, and then select **Add extension** if you wish to proceed.
5. You'll see a final prompt confirming the extension has been added.
