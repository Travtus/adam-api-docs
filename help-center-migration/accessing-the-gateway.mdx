# Accessing the Gateway

The Gateway can be accessed in one of a few ways:

## Google Chrome Extension

<a href="https://chrome.google.com/webstore" target="_blank">
  <img 
    src="/help-center-migration/images/chrome-web-store-badge.png" 
    alt="Available in the Chrome Web Store"
    noZoom 
    height="60"
  />
</a>

<Steps>
  <Step title="Visit the Chrome Web Store">
    Visit the Chrome Web Store or Travtus for Google Chrome
  </Step>
  <Step title="Search for Travtus">
    Search for "Travtus" in the Chrome Web Store
  </Step>
  <Step title="Install the Extension">
    When you find an extension you'd like to add, click **Add to Chrome**
  </Step>
</Steps>

## Outlook Extension

<a href="https://appsource.microsoft.com/" target="_blank">
  <img 
    src="/help-center-migration/images/microsoft-appsource-badge.png" 
    alt="Available in Microsoft AppSource"
    noZoom 
    height="60"
  />
</a>

<Steps>
  <Step title="Visit Microsoft AppSource">
    Visit Travtus in Microsoft AppSource
  </Step>
  <Step title="Get the App">
    Select **Get it now**
  </Step>
  <Step title="Install">
    Follow the instructions to download and install the app
  </Step>
</Steps>

## Microsoft Edge

<Steps>
  <Step title="Go to Chrome Web Store">
    In Microsoft Edge, go to the Chrome Web Store
  </Step>
  <Step title="Allow Extensions">
    Select **Allow extensions from other stores** in the banner at the top of the page, then select **Allow** to confirm
  </Step>
  <Step title="Select Extension">
    Select the extension you want to add and select **Add to Chrome**
  </Step>
  <Step title="Review Permissions">
    At the prompt showing permissions required by the extension, carefully review the permissions, and then select **Add extension** if you wish to proceed
  </Step>
  <Step title="Confirmation">
    You'll see a final prompt confirming the extension has been added
  </Step>
</Steps>
