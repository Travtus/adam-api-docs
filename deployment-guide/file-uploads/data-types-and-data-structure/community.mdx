---
title: "Communities"
---

Below is our standard data structure for a community:    

<ParamField body="Property" type="string" default="none">
  The name of the community to create.
</ParamField>

<ParamField body="Address" type="string" default="none">
  The first line of the address for the community you want to create.
</ParamField>

<ParamField body="city" type="string" default="none">
  The city of the address for the community you want to create.
</ParamField>

<ParamField body="state" type="string" default="none">
  The state of the address for the community you want to create.
</ParamField>

<ParamField body="ZIP" type="string" default="none">
  The postal code of the address for the community you want to create.
</ParamField>

<ParamField body="Adam Email Address" type="string" default="none">
  The email address for the community you want to create. Emails will be automatically ingested for the past 3 months for this email address.
</ParamField>

<ParamField body="Additional Email Addresses" type="string" default="none">
  Any additional O365 email inboxes to ingest emails from for the community to be created.
</ParamField>

<ParamField body="Unit Count" type="string" default="none">
  The number of units for the community to be created.
</ParamField>
